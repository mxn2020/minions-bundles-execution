# Welcome to Minions Bundle: Execution

We're excited to introduce **Minions Bundle: Execution** — Granular task management and daily tracking bundle — sprints, tasks, blockers.

## What's Included

- **TypeScript SDK**: `npm install @mxn2020/minions-bundles-execution-sdk`
- **Python SDK**: `pip install minions-bundles-execution`
- **CLI**: `npm install -g @mxn2020/minions-bundles-execution-cli`

Check out the [documentation](https://docs.minions.wtf) for a full guide.
