# Minions Bundle: Execution

Granular task management and daily tracking bundle — sprints, tasks, blockers

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@mxn2020/minions-bundles-execution-sdk`) — Core library for Node.js
- **Python SDK** (`minions-bundles-execution`) — Python bindings
- **CLI Tool** (`@mxn2020/minions-bundles-execution-cli`) — Command-line interface
- **Web App** — Interactive playground at [app.minions.wtf](https://app.minions.wtf)
