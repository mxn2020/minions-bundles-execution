# @mxn2020/minions-bundles-execution-cli

CLI for Minions Bundle: Execution — Granular task management and daily tracking bundle — sprints, tasks, blockers

## Installation

```bash
npm install -g @mxn2020/minions-bundles-execution-cli
```

## Usage

```bash
minions-bundles-execution --help
```
